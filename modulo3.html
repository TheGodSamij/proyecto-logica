Módulo 3 - Validación de Acceso

Este módulo incorpora el operador de negación para determinar si un usuario puede acceder al sistema.

S ↔ ((P ∧ Q) ∧ ¬R)
  • P: Usuario autenticado.
  • Q: Cuenta activa.
  • R: Cuenta bloqueada.
  • S: Acceso autorizado.

Usuario:

El acceso solo será autorizado si el usuario está autenticado, la cuenta está activa y no está bloqueada.
